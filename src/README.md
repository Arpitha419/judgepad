# JudgePad

JudgePad is a judging application built to streamline participant evaluation and score management.

## Features

- Add participants
- Assign scores
- Real-time score updates
- Clean and responsive UI
- Built using React

## Tech Stack

- React
- JavaScript
- CSS
- Vite
